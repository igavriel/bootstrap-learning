HTML
Hypertext Markup Language, a variation of XML that defines the structure of a webpage to the client displaying it.
CSS
Cascading Style Sheets, not exactly a language, but a list of instructions telling the HTML client how to style the layout defined in the HTML code.
JavaScript
The scripting language used inside of HTML documents to add interactivity and programmed features.
Style Sheet
A plain text file that is usually separate from the HTML document and holds CSS definitions to apply to the document.
