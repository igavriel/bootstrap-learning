Highlighted text is easy to achieve by using the mark tag. Again, semantics are important.

Using an "S" tag inside a paragraph will strike the text through, indicating that the text is no longer correct or accurate.

Using the "DEL" tag has a similar effect to "S", but the meaning is different, in that the text is to be or has been deleted.

The "SMALL" tag is mostly used for the small print, and will be liked by legal folks :-)

And "STRONG" is used to make text bolder so it stands out.

Whereas the "EM" tag is designed to emphasize text by making it appear italic.

The "U" tag will underline your text.

And finally, "INS" is designed to make it look like text has been inserted into a paragraph.
